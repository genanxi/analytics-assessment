## Visualizations

### Analysis

Analytical visualizations generated during the analysis process to support findings and recommendations. These may include correlation analysis, feature importance analysis, churn driver exploration, and other supporting charts.

### Dashboard Mockup

Conceptual dashboard designs demonstrating how analytical findings could be operationalized for stakeholders.

These mockups are not intended to be fully functional dashboards. Instead, they illustrate how the dashboard datasource layer created in `sql/03_dashboard_datasource.sql` could be leveraged to monitor customer health, retention risk, product adoption, and other key business metrics.
